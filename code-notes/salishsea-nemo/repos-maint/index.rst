*******************************************
NEMO and XIOS Code Repositories Maintenance
*******************************************

This section documents the maintenance processes for the NEMO and XIOS code respositories that are based on upstream Subversion repositories.

Researchers generally don't need to be concerned with the docs below.
They can follow the :ref:`QuickStartGuide` instructions for the system they are working on to clone the Mercurial code repositories they need from Bitbucket.
The docs below are about how those repos are created and kept in sync with the upstream Subversion repos.
